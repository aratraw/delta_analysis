
В рамках нашей библиотеки - ПРАВИЛЬНО:
       LazyRational acc, term;
        for (int i = 0; i < 80; ++i) {
             term = Sin(Rational(i)) * Cos(Rational(i + 1));//термы ленивые, НЕ ВЫЧИСЛЯЮТСЯ.
            acc + term;
        }
НЕПРАВИЛЬНО:
        LazyRational acc;
        for (int i = 0; i < 80; ++i) {
            LazyRational term = Sin(Rational(i).as_lazy()) * Cos(Rational(i + 1).as_lazy());//термы ленивые, НЕ ВЫЧИСЛЯЮТСЯ.
            acc + term;
        }
ИНИЦИАЛИЗИРОВАТЬ ПЕРЕМЕННУЮ ТЕРМА В ЦИКЛЕ - НЕПРАВИЛЬНО, потому что тогда для неё 80 раз вызовется конструктор и деструктор!
Неправильно: acc = acc+term;//МЫ ЗАПРЕТИЛИ ЭТО НА УРОВНЕ КОМПИЛЯТОРА, чтобы у вас НЕ БЫЛО ВОЗМОЖНОСТИ НАПИСАТЬ НЕОПТИМАЛЬНЫЙ КОД.
